# text-classifier

## What is scikit-learn?

scikit-learn is an open-source Python library used for classical machine learning (not deep learning). It provides ready-to-use implementations of classical models like regression, classification, clustering and preprocessing.

It's built on top of,
- numpy (array, math operations)
- scipy (advanced math, optimization)
- pandas (data handling, tables)

## How does scikit-learn works?

It follows a very consistent API design, almost entirely in 3 steps,

- initialize a model
- train a model (fit)
- use a model (predict or transform)
